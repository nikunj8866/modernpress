== Description ==

ModernPress is a versatile and modern WordPress theme designed with simplicity and functionality in mind. Whether you're running a personal blog, showcasing your portfolio, or building a business website, ModernPress provides all the tools you need to create a professional online presence.

**Key Features:**

* **Responsive Design**: Looks great on all devices - desktop, tablet, and mobile
* **Customizer Integration**: Easy-to-use theme customization options
* **Block Editor Ready**: Full support for the WordPress Block Editor (Gutenberg)
* **SEO Optimized**: Clean, semantic code that search engines love
* **Fast Loading**: Optimized for speed and performance
* **Accessibility Ready**: Built with accessibility best practices
* **Translation Ready**: Fully translatable and RTL language support
* **Custom Colors**: Choose your brand colors with the built-in color palette
* **Typography Options**: Beautiful typography with system fonts
* **Flexible Layout**: Multiple layout options with sidebar support
* **Social Media Integration**: Built-in social media menu support

**Perfect For:**
* Personal blogs
* Business websites
* Portfolio sites
* Creative agencies
* Freelancers
* Small businesses

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

== Frequently Asked Questions ==

= How do I customize the theme colors? =

Go to Appearance > Customize > Colors in your WordPress admin panel. ModernPress comes with a predefined color palette that you can customize to match your brand.

= Does this theme support custom menus? =

Yes! ModernPress supports custom menus. You can create and assign menus through Appearance > Menus in your WordPress admin.

= Is this theme translation ready? =

Absolutely! ModernPress is fully translation ready and includes RTL language support. You can translate it into any language using standard WordPress translation methods.

= Does the theme work with page builders? =

ModernPress is built primarily for the WordPress Block Editor (Gutenberg), but it should work with most popular page builders. However, for the best experience, we recommend using the native Block Editor.

= How do I set up the sidebar? =

The sidebar will automatically appear when you add widgets to it. Go to Appearance > Widgets and add widgets to the "Sidebar" widget area.
